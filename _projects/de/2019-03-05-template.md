# Verwendung der Vorlagen

Die Projekte werden in Markdown geschrieben. Markdown ist eine einfache Syntax, mit der sich Texte einfach in verschiedene Formate (z.B. HTML Code, PDF oder epub) konvertieren lassen. Diese Vorlage als Markdown Datei findest du [hier](https://raw.githubusercontent.com/sensebox/sensebox-reboot/master/_projects/de/2019-03-05-template.md) zum Download. Alternativ kannst du dein Projekt auch in folgender [Word-Datei](/docs/Formular_Projektidee.docx) dokumentieren und per E-Mail an info(at)sensebox.de senden.
